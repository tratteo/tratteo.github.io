Projects

    • Genetic trained Neural Networks

    • GitHub

    In this project, generations of dense Neural Networks (Multilayer Perceptrons) are trained using a Genetic Algorithm and a specific Fitness Function (ff).
    In the specific case, the NNs are applied to a system of self-driving cars, developed in Unity.
    Using different tracks to train the NNs allows the generations to adapt to different paths, generating a learning process that aims to the specific ability to drive, regardless of the path and track
