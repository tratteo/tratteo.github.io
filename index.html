Projects

    • Genetic trained Neural Networks

    • GitHub

    In this project, generations of dense Neural Networks (Multilayer Perceptrons) are trained using a Genetic Algorithm and a specific Fitness Function (ff).
    In the specific case, the NNs are applied to a system of self-driving cars, developed in Unity.
    Using different tracks to train the NNs allows the generations to adapt to different paths, generating a learning process that aims to the specific ability to drive, regardless of the path and track

    Features:

    • Unity implementation of basic car physics (wheels, suspensions)

    • Multidimensional arrays representation of Neural Networks

    • Neural Network DNA encoding and implementation for the Genetic Algorithm

    • Genetic Algorithm training process

    • Unity serialization of the fittest Neural Network DNA

    • Various Unity simulation scenes with multiple tracks

    • IOT Home Server

    • GitHub

    IOT multithreaded server for home domotic project.
    Representing the central hub that redirects all the clients requests to the designated devices.

    Features:

    • Classical Java socket communication

    • Parallel management of incoming connections

    • Customized UDP protocol in order to efficiently send webcam images to the clients connected that requested the streaming service

    • Communication with Arduino through serial port

    • Communication with Raspberry through wifi (sockets)

    • Multithreaded management of services, such as temperature thread, clock thread and logging thread
